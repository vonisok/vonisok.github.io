# Book Expansion Shortlist

This is the approval list before downloading any new PDFs or creating new book entries/thumbnails.

I only included titles that appear to have a clearly legitimate free source:

- public domain via Project Gutenberg
- official government PDF via NSA or CIA

## Recommended First Batch

These are the strongest fits for the current collection and the safest to automate.

### 1. Simple Sabotage Field Manual

- Fit: `Nerds / OPSEC` or `History / Intelligence`
- Why it fits: classic OSS sabotage manual, strong overlap with tradecraft / operational history
- Source status: official CIA PDF
- Source: [CIA PDF](https://www.cia.gov/static/5c875f3ec660e092cf893f60b4a288df/SimpleSabotage.pdf)

- Fit: `Nerds / Crypto`

- Source status: official NSA PDF
